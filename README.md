# 🎮 Slide Number Game

Slide Number Game is a simple and interactive puzzle game where players arrange shuffled numbers in the correct order by sliding tiles. This project is built using basic web technologies and is ideal for beginners to understand DOM manipulation and game logic.

## 🧠 How the Game Works

- The board contains numbered tiles with one empty space.
- Player can slide adjacent tiles into the empty space.
- The goal is to arrange numbers in the correct order.
- Game ends when all numbers are placed correctly.

---

## 🎮 How to Play

1. Start the game.
2. Click on tiles adjacent to the empty space.
3. Move tiles step-by-step.
4. Arrange all numbers in correct order.
5. Complete the puzzle to win 🎉
