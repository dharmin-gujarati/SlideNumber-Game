
# 🎯 SlideNumber Game (iOS)

A simple and interactive **number sliding puzzle game** built for iOS using Swift.  
The goal of the game is to arrange shuffled numbered tiles into the correct order by sliding them into the empty space.

---

## 📱 Features

- 🔢 Classic sliding number puzzle gameplay
- 🎮 Smooth tile animations
- 📊 Move counter tracking
- ⏱️ Timer to measure performance
- 🔄 Shuffle / Restart functionality
- 📐 Responsive layout for all iPhone sizes
- 🌙 Clean and minimal UI

---

## 🧠 Gameplay

- The board contains numbered tiles and one empty space.
- Tap on a tile adjacent to the empty space to move it.
- Arrange all numbers in ascending order to win the game.
- Try to solve in minimum moves and time!

---

## 🛠️ Tech Stack

- Language: **Swift**
- UI Framework: **UIKit / SwiftUI**
- Architecture: **MVC / MVVM (based on implementation)**
- Xcode Version: **14+ recommended**
